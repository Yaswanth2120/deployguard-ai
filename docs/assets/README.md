# docs/assets

Store walkthrough and portfolio screenshots here.

Follow the naming convention and redaction rules in
[../screenshot-checklist.md](../screenshot-checklist.md):

- leading two-digit order number (`01-...`, `02-...`)
- lowercase kebab-case filenames, `.png` preferred
- never commit real API keys, tokens, or non-local credentials in an image

This folder is currently a placeholder. Add the images listed in the
screenshot checklist as you capture them, then reference them from the docs
with relative paths, e.g. `![Dashboard](assets/01-dashboard-overview.png)`.
